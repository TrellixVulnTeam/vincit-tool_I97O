## Prerequisite
1. python3
2. pip3
3. OpenCV (version 4.5.5)
4. Pytesseract (version 5.1.0)

`python3 install OpenCV`
`python3 install pytesseract`

and 

`pip3 install OpenCV`
`pip3 install pytesseract`

## How to run this project :
Back-End :
1. cd vincit-rool/
2. npm install
3. npm start

Front-End :
1. cd vincit-tool/react-app/
2. npm install
3. npm start
